Settings will persisst across sessions.

Settings will be stored in ${HOME}/.config/${organization}/${application}.conf on Linux systems.

Example:

Parameters p("HephaestusVision", "HephaestusScanner"); p.setDefault("Height","1"); p.setDefault("Width","1"); p.show(); p.raise(); p.activateWindow(); .../// int height = p.getParameter("Height").toInt(); int wifht = p.getParameter("Width").toInt();
