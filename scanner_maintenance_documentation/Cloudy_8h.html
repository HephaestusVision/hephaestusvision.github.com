pcl::PointCloud<pcl::PointXYZRGBNormal>* vtkPolyData_to_point_cloud ( vtkPolyData *  poly)

Function to convert from vtkPolyData to a pcl::PointCloud<pcl::PointXYZRGBNormal>. Will get color information from the rgb_colors array if present.

The returned pointer will point to a object created on the stack. it is the caller's repsonsibility to delete the object.

Returns NULL on an error.
