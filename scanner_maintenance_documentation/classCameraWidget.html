Detailed Description

The CameraWidget class can be used where a QWidget can be used. It will talk to a camera using the open-cv and open-kinect libraries.

Superclass : http://qt-project.org/doc/qt-4.8/QGLWidget.html

To find out whether the camera is in working order, check the isGood() method.
