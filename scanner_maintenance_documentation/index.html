Plus a header file containing non-class routines:

  • MidasUpload.h
  • Cloudy.h

The program is structured as a QT application. As such, the classes DepthScannerMainWindow, QVTKPolyViewWidget, CameraWidget, LoginDialog, PickFolder, and Parameters are subclasses of QT Widgets that do specific things.

The MidasFolder and classe is little more than structs inteded to hold related information together.

Cloudy holds all of the computer-vision algorithms, as well as routines to convert from Point Cloud Library data structures to VTK data structures.

MidasUpload.h holds the functions for talking to a MIDAS server.
